void weight_manager_init ( QueueHandle_t hx711_queue,
calibration_t * calib )

Initialize the weight manager.

Parameters
hx711_queueQueueHandle_t from hx711_init_rtos()
calibPointer to calibration_t struct for conversion

Spawns a FreeRTOS task that:

  • Sleeps (no output) until weight ≥ WM_WAKE_THRESHOLD_G
  • On wake, prints weights and runs for WM_ACTIVE_TIMEOUT_MS
  • Returns to sleep on weight removal (< WM_SLEEP_THRESHOLD_G) or timeout
